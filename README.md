# CI/CD Entrega Final - Evidencia Sub 5

Este proyecto es una aplicación básica de Node.js que define una función matemática simple y sus respectivas pruebas unitarias con Jest. Está diseñado como un proyecto de demostración o evidencia para flujos de Integración Continua (CI).

## Estructura del Proyecto

- [app.js](file:///c:/Users/carre/Documents/workspace/ic-entrega-final-evidencia-sub5/app.js): Contiene la lógica principal, definiendo y exportando una función para sumar dos números.
- [app.test.js](file:///c:/Users/carre/Documents/workspace/ic-entrega-final-evidencia-sub5/app.test.js): Define pruebas automatizadas para la función de suma utilizando el framework Jest.
- [package.json](file:///c:/Users/carre/Documents/workspace/ic-entrega-final-evidencia-sub5/package.json): Configuración del proyecto Node.js, dependencias y scripts de ejecución.

## Requisitos

- [Node.js](https://nodejs.org/) (versión 14 o superior recomendada)
- npm (instalado por defecto con Node.js)

## Instalación

Instala las dependencias del proyecto ejecutando:

```bash
npm install
```

## Pruebas

Para ejecutar la suite de pruebas unitarias con Jest, utiliza el comando:

```bash
npm test
```
